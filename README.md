# PDF-Zusammenfugen Homebrew Tap

Offizieller Homebrew Tap fuer **pdf-zusammenfugen**.

## Installation

```bash
brew tap wsgtcyx/pdfzus https://github.com/wsgtcyx/PDF-Zusammenfugen-homebrew.git
brew install wsgtcyx/pdfzus/pdf-zusammenfugen
```

## Upgrade

```bash
brew update
brew upgrade wsgtcyx/pdfzus/pdf-zusammenfugen
```

## Deinstallation

```bash
brew uninstall wsgtcyx/pdfzus/pdf-zusammenfugen
brew untap wsgtcyx/pdfzus
```
